# <a name="appendix-a-azure-rtos-netx-secure-dtls-returnerror-codes"></a>Приложение A. Коды возврата и ошибок ОСРВ Azure NetX Secure DTLS

## <a name="netx-secure-tlsdtls-return-codes"></a>Коды возврата NetX Secure TLS или NetX Secure DTLS

Все коды TLS начинаются с 0x101, а значения кодов TCP/IP или UDP не превышают 0x100. Возвращаемые значения X.509 начинаются с 0x181. Сведения о возвращаемых значениях TCP/IP и UDP см. в документации по протоколам TCP/IP и UDP в NetX, а сведения о значениях X.509 приводятся ниже.

## <a name="netx-secure-x509-return-codes"></a>Коды возврата NetX Secure X.509
