# <a name="appendix-c---guix-widget-styles"></a>Приложение В. Стили мини-приложений в GUIX

__***Общие стили (используются с большинством типов мини-приложений):***__

**GX_STYLE_BORDER_NONE**
  - Значение: 0x00000000
  - Описание: используйте этот стиль для отрисовки мини-приложения без границы.

**GX_STYLE_BORDER_RAISED**
  - Значение: 0x00000001
  - Описание: отрисовка мини-приложения с приподнятой границей.

**GX_STYLE_BORDER_RECESSED**
  - Значение: 0x00000002
  - Описание: отрисовка мини-приложения с утопленной границей.

**GX_STYLE_BORDER_THIN**
  - Значение: 0x00000004
  - Описание: отрисовка границы шириной один пиксель.

**GX_STYLE_BORDER_THICK** 
  - Значение: 0x00000008
  - Описание: отрисовка мини-приложения с толстой границей.

**GX_STYLE_BORDER_MASK**
  - Значение: 0x0000000f
  - Описание: значение маски, используемое для тестирования только полей стиля элемента стиля мини-приложения.

**GX_STYLE_TRANSPARENT**
  - Значение: 0x10000000
  - Описание: создание, по крайней мере частично, прозрачного мини-приложения. Этот стиль следует использовать, когда мини-приложение не отрисовывается как полностью непрозрачное, в том числе если в качестве фона мини-приложения используется полупрозрачная пиксельная карта. Этот флаг стиля сообщает GUIX о том, что родительское мини-приложение должно обновить область фона мини-приложения.

**GX_STYLE_DRAW_SELECTED**
  - Значение: 0x20000000
  - Описание: указывает, что мини-приложение должно быть отрисовано с использованием цветов и шрифтов состояния выбора. Различные типы мини-приложений используют стиль DRAW_SELECTED по-разному, чтобы указать, что мини-приложение в настоящее время выбрано.

**GX_STYLE_ENABLED**
  - Значение: 0x40000000
  - Описание: мини-приложение помечается как включенное, что позволяет ему принимать события пользовательского ввода и формировать выходные сигналы.
  
**GX_STYLE_DYNAMICALLY_ALLOCATED**
  - Значение: 0x80000000
  - Описание: указывает, что память блока управления мини-приложения выделяется динамически с помощью службы gx_system_memory_allocator при создании мини-приложения и высвобождается при его уничтожении.

**GX_STYLE_USE_LOCAL_ALPHA**
  - Значение: 0x01000000
  - Описание: предписывает функциям рисования GUIX использовать локальный альфа-фактор мини-приложения при его отрисовке. Этот флаг обычно используется внутренней логикой GUIX для реализации анимации исчезания мини-приложения.


__***Стили выравнивания текста (применяются ко всем мини-приложениям, в которых отрисовывается текст):***__

**GX_STYLE_TEXT_LEFT**
  - Значение: 0x00001000
  - Описание: текст отрисовывается с выравниванием по левому краю в клиентской области мини-приложения.

**GX_STYLE_TEXT_RIGHT** 
  - Значение: 0x00002000
  - Описание: текст отрисовывается с выравниванием по правому краю в клиентской области мини-приложения.

**GX_STYLE_TEXT_CENTER**
  - Значение: 0x00004000
  - Описание: текст отрисовывается с выравниванием по центру в клиентской области мини-приложения.

**GX_STYLE_TEXT_COPY**
  - Значение: 0x00008000
  - Описание: по умолчанию мини-приложения, которые отрисовывают текст, сохраняют только указатель на текст, передаваемый приложением. Для статически определенного текста в таблице строк мини-приложению нет смысла создавать частную копию назначенного текста. Однако если текст, назначенный мини-приложению, создается динамически с помощью таких функций, как sprint() или gx_utility_ltoa, то часто бывает удобно, чтобы мини-приложение сохраняло собственную частную копию такого текста. Это позволяет приложению использовать автоматические или временные переменные при определении текстовых строк. В противном случае ему потребовалось бы определять статические символьные массивы для каждого текстового мини-приложения, использующего динамически определяемый текст. Если этот флаг стиля установлен, мини-приложение будет использовать функцию gx_system_memory_allocator для динамического выделения блока памяти, необходимого для хранения частной копии назначенной строки. Поэтому этот флаг стиля должен использоваться в приложении, в котором определены функции memory_allocator и memory_deallocator. Флаг GX_STYLE_TEXT_COPY не следует снимать после установки, так как это приведет к непредсказуемым результатам.

__***Стили кнопок (применяются только к типам мини-приложений с кнопками GUIX):***__

**GX_STYLE_BUTTON_PUSHED**
  - Значение: 0x00000010
  - Описание: указывает, что кнопка находится в состоянии нажатия или выбора.

**GX_STYLE_BUTTON_TOGGLE**
  - Значение: 0x00000020
  - Описание: кнопка меняет состояние с нажатого на отжатое и наоборот при каждом событии щелчка мышью. Этот стиль обычно используется с кнопками типа "флажок".

**GX_STYLE_BUTTON_RADIO**
  - Значение: 0x00000040
  - Описание: этот стиль указывает, что кнопка может быть единственной выбранной; при ее выборе отменяется выбор других кнопок в группе. Этот стиль обычно используется с кнопками типа "переключатель".

**GX_STYLE_BUTTON_EVENT_ON_PUSH**
  - Значение: 0x00000080
  - Описание: указывает, что кнопка создает событие щелчка мышью при первоначальном нажатии. По умолчанию событие щелчка мышью создается при отпускании кнопки.

**GX_STYLE_BUTTON_REPEAT**
  - Значение: 0x00000100
  - Описание: указывает, что кнопка должна непрерывно отправлять события щелчка мыши родительскому элементу, когда она удерживается в нажатом состоянии.

__***Стили списка (применяются только к мини-приложениям GUIX списочного типа):***__

**GX_STYLE_CENTER_SELECTED** 
  - Значение: 0x00000010
  - Описание: зарезервировано

**GX_STYLE_WRAP**
  - Значение: 0x00000020
  - Описание: дочерние элементы списка переносятся из начала в конец, когда список перетаскивается или прокручивается за пределы начального или конечного индекса списка.

**GX_STYLE_FLICKABLE**
  - Значение: 0x00000040
  - Описание: зарезервировано

__***Стили кнопок "пиксельная карта" и "значок":***__

**GX_STYLE_HALIGN_CENTER**
  - Значение: 0x00010000
  - Описание: пиксельная карта кнопки должна быть выровнена по центру горизонтальной оси в пределах границы кнопки.

**GX_STYLE_HALIGN_LEFT**
  - Значение: 0x00020000
  - Описание: пиксельная карта кнопки должна быть выровнена по левому краю горизонтальной оси в пределах границы кнопки.

**GX_STYLE_HALIGN_RIGHT**
  - Значение: 0x00040000
  - Описание: пиксельная карта кнопки должна быть выровнена по правому краю горизонтальной оси в пределах границы кнопки.

**GX_STYLE_VALIGN_CENTER**
  - Значение: 0x00080000
  - Описание: пиксельная карта кнопки должна быть выровнена по центру вертикальной оси в пределах границы кнопки.

**GX_STYLE_VALIGN_TOP**
  - Значение: 0x00100000
  - Описание: пиксельная карта кнопки должна быть выровнена по верхнему краю вертикальной оси в пределах границы кнопки.

**GX_STYLE_VALIGN_BOTTOM**
  - Значение: 0x00200000
  - Описание: пиксельная карта кнопки должна быть выровнена по нижнему краю вертикальной оси в пределах границы кнопки.

__***Стили ползунка (применяются только к GX_SLIDER и производным типам мини-приложений):***__

**GX_STYLE_SHOW_NEEDLE**
  - Значение: 0x00000200
  - Описание: этот стиль необходимо добавить для ползунка для отрисовки индикатора стрелки. Его можно отключить, если приложение должно отключить стрелку ползунка или нарисовать особый индикатор стрелки.

**GX_STYLE_SHOW_TICKMARKS**
  - Значение: 0x00000400
  - Описание: если этот стиль включен, мини-приложение ползунка будет выполнять программную отрисовку пунктирных линий делений.

**GX_STYLE_SLIDER_VERTICAL**
  - Значение: 0x00000800
  - Описание: установите этот флаг стиля, чтобы создать вертикальный ползунок; снимите его, чтобы создать горизонтальный ползунок.

__***Стили спрайтов (применяются только к типам мини-приложений GX_SPRITE):***__

**GX_STYLE_SPRITE_AUTO**
  - Значение: 0x00000010
  - Описание: указывает, что анимация спрайта будет запускаться автоматически, когда мини-приложение спрайта получает событие GX_EVENT_SHOW.

**GX_STYLE_SPRITE_LOOP**
  - Значение: 0x00000020
  - Описание: при использовании этого стиля мини-приложение спрайта будет поочередно отображать кадры анимации спрайта до тех пор, пока спрайт не будет остановлен приложением.

__***Стили ползунка на основе пиксельной карты:***__

**GX_STYLE_TILE_BACKGROUND**
  - Значение: 0x00001000
  - Описание: фоновое изображение ползунка мозаично заполняет ограничивающий прямоугольник спрайта. Это позволяет использовать небольшое вертикальное или горизонтальное изображение для заполнения фона ползунка.

__***Дополнительные стили индикатора выполнения:***__

**GX_STYLE_PROGRESS_PERCENT**
  - Значение: 0x00000010
  - Описание: когда задан этот стиль, на индикаторе выполнения отображается процентное значение, а не абсолютное. Текст выравнивается по центру ограничивающего прямоугольника индикатора выполнения.

**GX_STYLE_PROGRESS_TEXT_DRAW**
  - Значение: 0x00000020
  - Описание: текущее значение индикатора выполнения отрисовывается в виде десятичного числа по центру индикатора.

**GX_STYLE_PROGRESS_VERTICAL**
  - Значение: 0x0000040
  - Описание: указывает, что индикатор выполнения имеет вертикальную ориентацию. По умолчанию используется горизонтальная ориентация.

**GX_STYLE_PROGRESS_SEGMENT_FILL**
  - **Значение**: 0x00000100
  - Описание: значение индикатора выполнения обозначается в виде сегментированных прямоугольников с заливкой, а не в виде сплошной заливки.

__***Дополнительные стили кругового индикатора выполнения:***__

**GX_STYLE_RADIAL_PROGRESS_ALIAS**
  - Значение: 0x00000200
  - Описание: круговой индикатор выполнения отрисовывается с использованием стилей кисти со сглаживанием. Для этого требуется больше ресурсов ЦП, но индикатор выглядит лучше. На устройствах с низкой производительностью ЦП снятие этого флага стиля приведет к ускорению отрисовки.

**GX_STYLE_RADIAL_PROGRESS_ROUND**
  - Значение: 0x00000400
  - Описание: использование стиля кисти со скругленными концами линии при отрисовке дуги кругового индикатора выполнения. По умолчанию используются квадратные концы линий.

__***Дополнительные стили ввода текста:***__

**GX_STYLE_ CURSOR_BLINK**
  - Значение: 0x00000040
  - Описание: курсор мини-приложения ввода текста будет мигать.

**GX_STYLE_ CURSOR_ALWAYS**
  - Значение: 0x00000080
  - Описание. Курсор мини-приложения ввода текста обычно отображается, только когда мини-приложение получает фокус ввода. Этот флаг стиля делает курсор видимым всегда независимо от фокуса ввода.

**GX_STYLE_TEXT_INPUT_NOTIFY_ALL**
  - Значение: 0x00000100
  - Описание: когда установлен этот флаг стиля, событие GX_EVENT_TEXT_EDITED создается при каждом получении события нажатия клавиши в мини-приложении ввода текста.

__***Дополнительные стили окна:***__

**GX_STYLE_TILE_WALLPAPER**
  - Значение: 0x00040000
  - Описание: клиентский прямоугольник окна мозаично заполняется назначенным изображением обоев.

**GX_STYLE_AUTO_HSCROLL**
  - Значение: 0x00100000
  - Описание: зарезервировано для использования в будущем.

**GX_STYLE_AUTO_VSCROLL**
  - Значение: 0x00200000
  - Описание: зарезервировано для использования в будущем.

__***Дополнительные стили меню:***__

**GX_STYLE_MENU_EXPANDED**
  - Значение: 0x00000010
  - Описание: мини-приложение меню типа "гармошка" изначально находится в развернутом состоянии.

__***Дополнительные стили представления в виде дерева:***__

**GX_STYLE_TREE_VIEW_SHOW_ROOT_LINES**
  - Значение: 0x00000010
  - Описание: мини-приложение представления в виде дерева должно рисовать линии от значка узла к корневому узлу дерева.

__***Дополнительные стили полосы прокрутки:***__

**GX_SCROLLBAR_BACKGROUND_TILE**
  - Значение: 0x00010000
  - Описание: зарезервировано для использования в будущем.

**GX_SCROLLBAR_RELATIVE_THUMB**
  - Значение: 0x00020000
  - Описание: ширина горизонтальной или высота вертикальной полосы прокрутки вычисляется на основе отношения видимой области родительского окна к минимальному и максимальному диапазону полосы прокрутки.

**GX_SCROLLBAR_END_BUTTONS**
  - Значение: 0x00040000
  - Описание: полоса прокрутки автоматически создает и прикрепляет кнопки в каждом конце области полосы прокрутки.

**GX_SCROLLBAR_VERTICAL** 
  - Значение: 0x01000000
  - Описание: для полосы прокрутки используется вертикальная ориентация.

**GX_SCROLLBAR_HORIZONTAL**
  - Значение: 0x02000000
  - Описание: для полосы прокрутки используется горизонтальная ориентация.

__***Стили текста колеса прокрутки:***__

**GX_STYLE_TEXT_SCROLL_WHEEL_ROUND**
  - Значение: 0x00000200
  - Описание: для отображения колеса прокрутки круглой формы используется синусоидальный алгоритм. Этот флаг стиля может создать дополнительную нагрузку на мини-приложение типа "колесо прокрутки", но также может придать ему реалистичный трехмерный вид.
